# 简介

[birdengcopy]: http://www.birdcopy.com

Birdcopy是菜鸟微云提供的一个手机应用,旨在帮助小微社群提供文件管理云平台。
关于菜鸟微云的介绍请见[Birdcopy 官网][birdengcopy]。

## 特点

主要功能
- 支持常见文档和音视频的在线观看和离线下载
- 支持虚拟金币的使用
- 支持聊天功能（融云）

支持文档类型
- PDF(+Mp3背景)
- Mp3（＋字幕） 类似听歌软件的表现形式
- Mp4

支持内容管理
- 标签化分类、搜索、展示
- 封面内容推荐
- 频道内容推荐

- Simple interface
- Works on both Andorid（11+） & iOS（6.0+）
